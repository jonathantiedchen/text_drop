# Text Drop

A zero-friction scratchpad for the browser. No accounts, no saving, no formatting toolbars — just a place to dump text while you work.

## What it does

- Full-screen text area that's ready to type the moment you open it
- **Copy all** grabs everything to your clipboard
- **Clear** wipes the slate
- Tab key inserts a tab (doesn't jump focus)
- Nothing is persisted — close the tab and it's gone

## Hosting

It's a single `index.html` file. Host it anywhere:

```bash
# GitHub Pages — push to a repo, enable Pages in settings

# Netlify / Vercel — drag and drop the file

# Local — just open it in your browser
open index.html
```
